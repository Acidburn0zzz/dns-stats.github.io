Welcome to dns-stats Home Page

dns-stats is a covering entity for the implementation of the Hedgehog DNS statistics presenter and related developments. Hedgehog is an alternative to the DSC presenter application which can be used to display real time statistics for DNS traffic.

Hedgehog was developed for ICANN by Sinodun IT and is released open source under an Apache2 license.

  • Read more about the Hedgehog software here: About Hedgehog.

  • The full website for the Hedgehog project is available here: Hedgehog Homepage

  • Github site for Hedgehog is here: Hedgehog on Github
